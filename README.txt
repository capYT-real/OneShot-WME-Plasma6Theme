OneShot World Machine Edition
game by Future Cat Games
theme by capYT

theme based on Commonality gris Gris by Mark Whittaker
https://www.opencode.net/phob1an/commonality

to install:
move "OneShot aurorae theme" to "~/.local/share/aurorae/" and rename it to "OneShot"
move "OneShot desktoptheme" to "~/.local/share/plasma/desktoptheme/" and rename it to "OneShot"
move "OneShot look-and-feel" to "~/.local/share/plasma/look-and-feel/" and rename it to "OneShot"
in settings > Global Theme > Colors click "Install from file" and select "OneShot.colors"
in settings > Text & Fonts > Font Management click "Install from file" and select "TerminusTTF-Bold.ttf"
if you want a OS WME wallpaper, right click the provided image and set it as wallpaper, or search OneShot on The Spriters Resource website
if you know how to make cursors for linux, an image file has been provided for that to, cursor sprites also by Future Cat Games with some by capYT
